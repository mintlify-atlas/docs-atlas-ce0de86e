---
title: 'Save Bookmark'
description: 'Create a new bookmark with optional category assignment'
---

## Endpoint

```
POST /link/save
```

Save a new bookmark to your collection. If you specify a category that doesn't exist, it will be automatically created. If no category is provided, the bookmark will be saved under "no category".

## Request Body

<ParamField body="url" type="string" required>
  The URL of the bookmark you want to save
</ParamField>

<ParamField body="Name" type="string" required>
  The display name for your bookmark
</ParamField>

<ParamField body="Description" type="string">
  An optional description of the bookmark. This field can be omitted or left empty.
</ParamField>

<ParamField body="categoryName" type="string">
  The category name to organize your bookmark. If empty or omitted, defaults to "no category". If the category doesn't exist, it will be created automatically.
</ParamField>

## Response

<ResponseField name="message" type="string">
  A success message indicating the operation result. Returns either:
  - "link saved successfully" - when using an existing category
  - "link and category saved successfully" - when a new category was created
</ResponseField>

<ResponseField name="save" type="object">
  The created bookmark object
  
  <Expandable title="properties">
    <ResponseField name="id" type="number">
      The unique identifier for the bookmark
    </ResponseField>
    
    <ResponseField name="url" type="string">
      The saved URL
    </ResponseField>
    
    <ResponseField name="Name" type="string">
      The bookmark name
    </ResponseField>
    
    <ResponseField name="Description" type="string | null">
      The bookmark description, or null if not provided
    </ResponseField>
    
    <ResponseField name="CategoryId" type="number">
      The ID of the associated category
    </ResponseField>
    
    <ResponseField name="createdAt" type="string">
      ISO 8601 timestamp of when the bookmark was created
    </ResponseField>
  </Expandable>
</ResponseField>

<CodeGroup>

```bash cURL

  -H "Content-Type: application/json" \
```

```python Python
import requests

url = "https://square-forest-972c.yumnambilson.workers.dev/link/save"

    "url": "https://github.com",
    "Name": "GitHub",
```

## Behavior Notes

<Info>
  **Automatic Category Creation**: If you provide a `categoryName` that doesn't exist in the database, the API will automatically create it for you.
</Info>

<Info>
  **Default Category**: When `categoryName` is omitted or empty, the bookmark is saved under "no category".
</Info>

## Error Responses

| Status Code | Description |
|-------------|-------------|
| 201 | Bookmark saved successfully |
| 500 | Internal server error during save operation |
