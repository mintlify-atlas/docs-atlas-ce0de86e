## Endpoint

```
GET /link/showall
```

Retrieve all bookmarks from your collection. Each bookmark includes its associated category information.

## Request

This endpoint does not require any parameters or request body.

## Response

<ResponseField name="result" type="array">
  An array of bookmark objects. Returns an empty array if no bookmarks exist.
  
  <Expandable title="bookmark object">
    <ResponseField name="id" type="number">
      The unique identifier for the bookmark
    </ResponseField>
    
    <ResponseField name="url" type="string">
      The saved URL
    </ResponseField>
    
    <ResponseField name="Name" type="string">
      The bookmark name
    </ResponseField>
    
    <ResponseField name="Description" type="string | null">
      The bookmark description, or null if not provided
    </ResponseField>
    
    <ResponseField name="CategoryId" type="number">
      The ID of the associated category
    </ResponseField>
    
    <ResponseField name="createdAt" type="string">
      ISO 8601 timestamp of when the bookmark was created
    </ResponseField>
    
    <ResponseField name="category" type="object">
      The associated category information
      
      <Expandable title="properties">
        <ResponseField name="CategoryName" type="string">
          The name of the category
        </ResponseField>
      </Expandable>
    </ResponseField>
  </Expandable>
</ResponseField>

## Response Fields

The endpoint returns all bookmark information including:

- Basic bookmark data (id, url, Name, Description)
- Timestamp information (createdAt)
- Category relationship (CategoryId and nested category object)

<Info>
  The `category` object is nested within each bookmark, providing easy access to the category name without requiring a separate API call.
</Info>

## Error Responses

| Status Code | Description |
|-------------|-------------|
| 200 | Bookmarks retrieved successfully |
| 404 | No bookmarks found in the database |

## Use Cases

This endpoint is ideal for:

- Displaying all bookmarks in your application
- Exporting bookmark data
- Synchronizing bookmarks across devices
- Building search or filter functionality
