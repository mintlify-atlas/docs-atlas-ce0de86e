  <h2 className="text-2xl sm:text-3xl font-semibold text-gray-900 dark:text-white mb-3">
    Quick Start
  </h2>
  <p className="text-base text-gray-600 dark:text-gray-400 mb-8">
    Get up and running with Bookmark Deeploy in minutes
  </p>

  <Steps>
    <Step title="Clone the repository">
      Clone the Bookmark Deeploy repository and navigate to the project directory.
      
      ```bash
      git clone https://github.com/BilsoNyuMNAM/bookmarkDeeploy.git
      cd bookmarkDeeploy
      ```
    </Step>
    
    <Step title="Set up the database">
      Configure your PostgreSQL database with Prisma Accelerate and run migrations.
      
      ```bash
      cd Backend
      npm install
      npx prisma migrate deploy
      ```
      
      <Note>
        You'll need a Prisma Accelerate connection URL. See the [database setup guide](/guides/database-setup) for details.
      </Note>
    </Step>
    
    <Step title="Configure environment variables">
      Set up your environment variables for the backend worker.
      
      ```bash
      # Add to wrangler.toml or use wrangler secrets
      wrangler secret put DATABASE_URL
      ```
    </Step>
    
    <Step title="Start the development servers">
      Run the backend and frontend in development mode.
      
      ```bash
      # Terminal 1 - Backend
      cd Backend
      npm run dev
      
      # Terminal 2 - Frontend
      cd frontend
      npm install
      npm run dev
      ```
      
      Your application is now running! The frontend is available at `http://localhost:5173`.
    </Step>
  </Steps>
</div>
